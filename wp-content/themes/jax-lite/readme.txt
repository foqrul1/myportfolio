Jax Lite WordPress Theme
---------------------

Jax Lite is a minimal and creative WordPress theme, based on Bootstrap framework and optimized for mobile devices, is easy to customize and manage. Thanks to masonry layout, is possible to use Jax Lite, for create a beautiful photo showcase.

Created by ThemeinProgress, http://www.themeinprogress.com 
Demo: https://www.themeinprogress.com/jax-free-responsive-creative-wordpress-theme/
